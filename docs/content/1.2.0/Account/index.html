Account Management

Account management​

ไว้ใช้การจัดการบัญชี โดยมีการจัดการสองอย่าง ดังนี้

  • Accounts ไว้ใช้จัดการบัญชีของผู้ใช้
  • Role Setting จัดการบทบาทของผู้ใช้ในระบบ
  • Accounts
  • Role Settings

Accounts​

มีข้อมูลสำคัญดังนี้

  • Create a new account profile ไว้ใช้สร้างบัญชีผู้ใช้
  • Table แสดงข้อมูลบัญชีที่ทำการสร้างหรือแก้ไขไว้ล่าสุด
  • Action เพื่อใช้ในการจัดการบัญชี

Create a new account profile​

โดยข้อมูลที่จำเป็นต้องกรอกมีอยู่ 2 อย่างคือ

  • Email โดยกำหนดเป็น Format : exam@simple.com
  • Role โดยสามารถเลือกตำแหน่งอ้างอิงจากที่สร้างไว้ใน Role Setting นอกเหนือจากนี้จะเป็นข้อมูลที่กรอกหรือไม่กรอกก็ได้
  • First name ชื่อ
  • Last name นามสกุล
  • Position ตำแหน่ง
  • Phone number เบอร์โทรศัพท์ หลังจากกรอกข้อมูลที่จำเป็นเรียบร้อยแล้วให้กด Save เพื่อบันทึกข้อมูล

Table​

แสดงข้อมูลที่สร้างหรือแก้ไขไว้ดังนี้

  • Account holder แสดง First name / Last name
  • Email แสดง Email
  • Position แสดง Position
  • Role แสดง Role

Action​

ไว้ใช้ในการจัดการบัญชีผู้ใช้

  • View ไว้ใช้สำหรับตรวจสอบข้อมูล
  • Edit ไว้ใช้สำหรับแก้ไขข้อมูลต่างๆ
  • Delete ไว้ใช้สำหรับลบบัญชีที่ไม่ต้องการ

Role Settings​

มีข้อมูลสำคัญดังนี้

  • Add a new role ไว้ใช้สร้างบทบาทและกำหนดสิทธิในการเข้าถึง
  • Table แสดงชื่อบทบาทที่มีการสร้างหรือแก้ไข
  • Action ไว้ใช้จัดการบทบาทที่มีการสร้างขึ้น

Add a new role​

สิ่งที่จำเป็นต้องกรอกคือ

  • Role name ชื่อของบทบาท นอกนั้นจะเป็นสิทธิในการเข้าถึงต่างๆ สามารถเลือกหรือไม่เลือกได้เช่นกัน
  • Fucntion สิทธิการเข้าถึง

Table​

แสดงข้อมูลที่สร้างหรือแก้ไขไว้ดังนี้

  • Role แสดงชื่อบทบาทที่มีการสร้างหรือแก้ไขไว้

Action​

ไว้ใช้ในการจัดการบทบาท

  • View ไว้ใช้ในการตรวจสอบบทบาทที่มีการสร้างหรือแก้ไขล่าสุด
  • Edit ไว้ใช้ในการแก้ไขชื่อ หรือ สิทธิในการเข้าถึงของบทบาทนั้นๆ
  • Delete ไว้ใช้ในการลบบทบาทที่ไม่ต้องการออก
