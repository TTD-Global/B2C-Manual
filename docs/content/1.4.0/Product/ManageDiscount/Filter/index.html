Filter discount

ใช้ในการช่วยค้นหา Product จะแบ่งเป็น 3 ส่วนดังนี้

  • Search Product
  • Search options
  • Filter by status

Search Product ไว้ใช้ในการกรอกค้นหาชื่อ Product ที่ใช้กับโค้ดส่วนลด

Search Options ไว้ใช้ในการกรอกค้นหาชื่อ Options ที่ใช้กับโค้ดส่วนลด

Filter by status มี Status ดังนี้

  • All แสดง Status ของ Discount ที่มีทั้งหมด

  • Active แสดง Discount ที่มี Status : Active ทั้งหมด

  • Paused แสดง Discount ที่มี Status : Paused ทั้งหมด

  • Expired แสดง Discount ที่มี Status : Expired ทั้งหมด
