Tab Status

Tab Status จะมีทั้งหมด 4 Tab ด้วยกันดังนี้

  • All
  • Pending
  • Confirmed
  • Cancelled

All จะแสดง Status ทั้งหมดที่มีโดยเรียงวันที่จาก Booking date / วันที่ Booking เข้ามา

Pending เป็นสถานะที่รอดำเนินการ Confirm / Cancel Booking โดยจะมี 2 กรณีดังนี้

  • Pending เป็นสถานะที่รอดำเนินการ Confirm / Cancel Booking โดยที่ยังไม่เกินระยะเวลาเงื่อนไขของ Confirmation Policy

  • Overdue Pending เป็นสถานะที่เกินระยะเวลาเงื่อนไขของ Confirmation Policy ตัวอย่างกรณีต่างๆ
    • 24/48 hr. confirm Product1 ตั้งเงื่อนไข 24hr. confirmation หากเกิน 24 ชม.แล้ว ยังไม่มีการดำเนินการใดๆจากฝั่ง Admin ระบบจะทำการขึ้น Overdue ให้ Admin ทราบว่าเกินกำหนดแล้ว
    • 48/78 hr. before departure Product2 ตั้งเงื่อนไข 48hr. before confirmation จะขึ้น Overdue ช่วง 24 ชม. ก่อนที่จะเริ่มกิจกรรมหรือออกเดินทาง

  • Action Button/View จะสามารถ Confirm / Cancel Booking ได้

  • P.S. สามารถเลือก Check box เพื่อทำการ Confirm หลายๆ Booking ได้

Confirmed เป็นสถานะที่ได้ Confirm Booking ลูกค้าเรียบร้อย

  • Cancel Booing สถานะนี้ยังสามารถ Cancel Booking ได้

  • P.S. หากเป็นกรณี Instant confirm เมื่อลูกค้าชำระเงินเรียบร้อย ระบบจะ Confirm Booking ให้ทันทีไม่ต้องรอ Admin

Cancelled เป็นสถานะที่ได้ Cancel Booking ลูกค้า ไม่ว่าจะเป็นกรณียกเลิกด้วยตัวเองหรือ ยกเลิกด้วย Admin
