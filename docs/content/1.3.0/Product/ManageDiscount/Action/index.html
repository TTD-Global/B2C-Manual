Button Action

ใช้ในการจัดการ Discount ประกอบด้วย 4 ส่วนดังนี้

  • Edit
  • View
  • Pause-Reactivate
  • Delete

Edit มี 2 ส่วนที่สามารถแก้ไข Discount ได้ดังนี้

  • Basic infomation
  • Set up Product & options

Basic infomation แก้ไขข้อมูลเบื้องต้นของ Discount ซึ่งสามารถแก้ไขได้ 3 ส่วนด้วยกันดังนี้

  • Discount Name ชื่อ Discount (ที่แสดงเฉพาะหน้า Admin)

  • Usage Quantity ระบุจำนวน Discount ที่สามารถใช้ได้ หรือไม่จำกัด

  • Maximum number of codes per buyer ระบุจำนวนสูงสุดที่สามารถใช้ Discount ต่อผู้ใช้งาน 1 ท่าน

Set up Product & options สามารถเพิ่มหรือแก้ไข Product / Options ได้

View แสดงข้อมูล Discount Detail แบ่งเป็น 3 ส่วนดังนี้

  • Manage discount
  • Discount details
  • Purchase Orders

Manage discount การจัดการ Discount แบ่งเป็น 2 อย่างดังนี้

  • Pause-Reactive จัดการเปิดการใช้งาน / หยุดการใช้งานของ Discount

  • Delete discount ลบ Discount

Discount details แสดงข้อมูล Discount ดังนี้

  • Status แสดง Status 2 อย่างคือ Active และ Pause

  • Percent / Price Amount แสดงจำนวนที่ลดราคาเป็น Percent / Price Amount ตามที่ตั้งค่าไว้

  • Button Action กดเพื่อทำการ Edit ได้

  • Name ชื่อของ Discount (แสดงเฉพาะ Admin)

  • Code ชื่อ Code Discount สามารถกดเพื่อ Copies เพื่อไปใช้งานได้

  • Applies to แสดงจำนวน Product (หากตั้งค่าเป็น Selected Product จะแสดงชื่อ Product เมื่อ Hover)

  • Claiming Period ช่วงระยะเวลาในการใช้ Code Discount

  • Usage Quantity หากตั้งค่าเป็น Specific จะแสดงจำนวนสูงสุดที่สามารถใช้ Discount ได้ หากตั้งค่าเป็น No limit จะแสดง "No limit"

  • Usage หากตั้งค่าเป็น Specific จะแสดง "จำนวนผู้ใช้โค้ด / จำนวนสูงสุดที่สามารถใช้ Discount ได้" หากตั้งค่าเป็น No limit จะแสดงจำนวนผู้ใช้ Discount

Purchase Orders แสดงตารางข้อมูล Booking ที่ Discount ได้ถูกใช้ไปโดยแสดงข้อมูลดังนี้

  • Booking date แสดงวันที่ที่ทำการ Booking โดยเรียงจากวันที่ล่าสุด

  • Booking ID แสดงหมายเลข Booking โดยสามารถกดที่หมายเลขเพื่อตรวจสอบข้อมูล Booking ได้

  • Product แสดงชื่อ Product และ Options ที่ใช้ Discount นี้ โดยสามารถกดที่ชื่อ Product เพื่อตรวจสอบข้อมูล Product details ได้

  • Detail แสดงจำนวนคนที่ลูกค้าเลือก และแสดงราคาเต็มพร้อมจำนวนที่ลูกค้าจ่ายหลังจากใช้ Discount

  • User แสดงชื่อและ Email ผู้ใช้งาน Discount

Pause-Reactivate ปรับเปลี่ยนสถานะที่ใช้งานอยู่เพื่อเปิด-หยุดการใช้งานของ Discount

Delete ลบ Discount ออกจากระบบ
