  • Pause - Reactivate - หยุด/เปิดใช้งาน Discount
  • Delete Discount - ลบ Discount

  • Status - จะแสดงสถานะ 3 สถานะ ดังนี้
    • Active - กำลังใช้งาน
    • Paused - หยุดใช้งาน
    • Expired - หมดอายุ
  • Action - สามารถแก้ไข Discount หรือดูรายละเอียด Discount ได้
  • รายละเอียด Discount - แสดงข้อมูลสำคัญของ Discount ดังนี้
    • Name - ชื่อของ Discount
    • Code - รหัสสำหรับใช้รับส่วนลด (สามารถกดเพื่อคัดลอกได้)
    • Applies to - ระบุว่าสามารถใช้กับสินค้าใดได้บ้าง
    • Claiming Period - ช่วงเวลาที่สามารถใช้ Discount ได้
    • Usage Quantity - จำนวนครั้งที่สามารถใช้ Discount ได้ (No limit หมายถึงไม่จำกัดจำนวนครั้ง)
    • Usage - จำนวนครั้งที่มีการใช้ Discount นี้ไปแล้ว
    • Discount Amount - แสดงจำนวนเงินหรือเปอร์เซ็นต์ส่วนลด

  • รายละเอียดตาราง Purchase Orders
    • Booking date - วันที่ทำการจอง
    • Booking ID - หมายเลขการจอง สามารถคลิกเพื่อดูรายละเอียดการจองได้
    • Product - ชื่อ Product และ ชื่อ Options
    • Detail - รายละเอียดเพิ่มเติม เช่น จำนวนผู้เดินทาง ราคาหลังหักส่วนลด และราคาก่อนหักส่วนลด
    • User - อีเมลหรือชื่อผู้ใช้งานที่ทำการจองและใช้ Discount

  • Pause-Reactive - ใช้สำหรับเปิดใช้งาน - หยุดการใช้งานชั่วคราว

  • Delete - ลบ Discount
